Results Screen

There are two different results views:

  • Standard View
  • Advanced View
The Default view may be changed in the Options.

Standard View

  1. Radius
    The Radius that was entered previously.

  2. Entrance Length
    The Entrance Length that was entered previously.

  3. Budget
    Total Number of Bricks calculated for the Budget size (40 × 25 × 15cm).

  4. Superior
    Total Number of Bricks calculated for the Superior size (35 × 25 × 20cm).

  5. Luxury
    Total Number of Bricks calculated for the Luxury size (30 × 20 × 20cm).

  6. Save
    Saves the current results. Click here for details.

  7. Advanced View
    Switches the results screen to Advanced View.

  8. Help
    Displays the current Web Help.

  9. Close
    Closes the program.

  10. Back
    Returns to the previous screen to allow for new values to be entered.

    Note: This is disabled when loading from a previously saved file.

Advanced View

  1. Same as in Standard View
    Refer to Standard View.

  2. Budget / Superior / Luxury
    Switches between displaying values for the Budget, Superior and Luxury brick sizes respectively.

  3. Entrance Tunnel Length
    Length of the entrance tunnel in bricks.

  4. Total No. of Rows
    Total Number of Rows in the main dome.

  5. Total No. of Bricks for Entrance
    Total Number of Bricks required to make the entrance tunnel.

  6. Total No. of Bricks for Igloo Dome
    Total Number of Bricks required to make the main igloo dome.

  7. No. of Bricks per Row
    Number of Bricks required for each row of the main dome.

    Note: Numbers marked with an * signify that bricks have been removed to allow for an entrance hole.

  8. Save
    Saves the current results. Click here for details.

  9. Standard View
    Switches the results screen to Standar View.

  10. Help
    Displays the current Web Help.

  11. Close
    Closes the program.

  12. Back
    Returns to the previous screen to allow for new values to be entered.

    Note: This is disabled when loading from a previously saved file.
