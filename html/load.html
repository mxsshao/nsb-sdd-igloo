Load Results

  1. Click on the 'Load' button on the Results Screen.

  2. When the Load Dialog appears, navigate to the file to be opened then click 'Open'.

    Note: The default load location can be changed in the options. Click here for details.

  3. The 'Results Screen' will open after the file has been loaded.
